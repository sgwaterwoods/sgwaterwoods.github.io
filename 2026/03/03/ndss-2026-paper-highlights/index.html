NDSS 2026 论文精选 - 网络与分布式系统安全研讨会

NDSS 2026 (Network and Distributed System Security Symposium) 将于 2026 年 2 月 23-27 日在美国圣地亚哥举行。今年共接收了 265 篇论文 (夏季周期 113 篇，秋季周期 152 篇)，其中 194 篇将在会议期间展示。

Conference: NDSS 2026 - Network and Distributed System Security Symposium
Date: February 23-27, 2026
Location: San Diego, California
Total Accepted Papers: 265 (113 summer cycle + 152 fall cycle)
Presenting: 194 papers

一、LLM 安全 / LLM Security

1. A Causal Perspective for Enhancing Jailbreak Attack and Defense

Authors: Licheng Pan (Zhejiang University), Yunsheng Lu (University of Chicago), Jiexi Liu (Alibaba Group), et al.

English Abstract: This paper introduces a causal perspective to enhance both jailbreak attacks and defenses for large language models. The approach analyzes the causal relationships between input prompts and model outputs to identify vulnerabilities and strengthen security measures.

中文摘要: 本文从因果视角出发，增强大型语言模型的越狱攻击和防御能力。该方法分析输入提示和模型输出之间的因果关系，以识别漏洞并加强安全措施。

2. ACE: A Security Architecture for LLM-Integrated App Systems

Authors: Evan Li (Northeastern University), Tushin Mallick, Evan Rose, William Robertson, Alina Oprea, Cristina Nita-Rotaru

English Abstract: ACE proposes a comprehensive security architecture for applications that integrate large language models. The framework addresses unique security challenges arising from LLM integration, including data leakage, prompt injection, and unauthorized access to sensitive APIs.

中文摘要: ACE 提出了一个针对集成大型语言模型的应用程序的综合安全架构。该框架解决了 LLM 集成带来的独特安全挑战，包括数据泄露、提示注入和对敏感 API 的未授权访问。

3. Attention is All You Need to Defend Against Indirect Prompt Injection Attacks in LLMs

Authors: Yinan Zhong (Zhejiang University), Qianhao Miao, Yanjiao Chen, et al.

English Abstract: This work demonstrates that attention mechanisms alone can effectively defend against indirect prompt injection attacks in large language models, without requiring additional computational overhead or architectural modifications.

中文摘要: 本研究表明，仅使用注意力机制就可以有效防御大型语言模型中的间接提示注入攻击，无需额外的计算开销或架构修改。

4. Beyond Jailbreak: Unveiling Risks in LLM Applications Arising from Blurred Capability Boundaries

Authors: Yunyi Zhang (Tsinghua University), Shibo Cui, Baojun Liu, et al.

English Abstract: This research identifies security risks in LLM applications that arise from unclear boundaries between different model capabilities, leading to unintended information leakage and privilege escalation.

中文摘要: 本研究识别了 LLM 应用中因不同模型能力之间界限模糊而产生的安全风险，导致意外的信息泄露和权限提升。

5. Bleeding Pathways: Vanishing Discriminability in LLM Hidden States Fuels Jailbreak Attacks

Authors: Yingjie Zhang (Institute of Information Engineering, CAS), Tong Liu, Zhe Zhao (Ant Group), et al.

English Abstract: The paper reveals how the vanishing discriminability in LLM hidden states creates vulnerabilities that enable jailbreak attacks, providing both theoretical analysis and practical attack demonstrations.

中文摘要: 本文揭示了 LLM 隐藏状态中消失的可辨别性如何产生允许越狱攻击的漏洞，提供了理论分析和实际攻击演示。

6. Cache Me, Catch You: Cache Related Security Threats in LLM Serving Frameworks

Authors: XiangFan Wu (Ocean University of China), Lingyun Ying (QI-ANXIN), et al.

English Abstract: This work uncovers cache-related security vulnerabilities in LLM serving frameworks that can be exploited to leak sensitive information across different user sessions.

中文摘要: 本研究揭示了 LLM 服务框架中与缓存相关的安全漏洞，这些漏洞可被利用在不同用户会话之间泄露敏感信息。

7. Character-Level Perturbations Disrupt LLM Watermarks

Authors: Zhaoxi Zhang (University of Technology Sydney), Xiaomei Zhang (Griffith University), et al.

English Abstract: The research demonstrates that character-level perturbations can effectively remove or disrupt watermarks embedded in LLM-generated text, raising concerns about content attribution and copyright protection.

中文摘要: 研究表明，字符级扰动可以有效地移除或破坏嵌入在 LLM 生成文本中的水印，引发了对内容归属和版权保护的担忧。

8. Characterizing the Implementation of Censorship Policies in Chinese LLM Services

Authors: Anna Ablove (University of Michigan), Shreyas Chandrashekaran, Xiao Qiang (UC Berkeley), Roya Ensafi

English Abstract: This empirical study characterizes how censorship policies are implemented in Chinese LLM services, revealing the techniques used to filter content and their effectiveness.

中文摘要: 这项实证研究描述了中国 LLM 服务中审查政策的实施方式，揭示了用于过滤内容的技术及其有效性。

9. Chasing Shadows: Pitfalls in LLM Security Research

Authors: Jonathan Evertz (CISPA), Niklas Risse (Max Planck), et al.

English Abstract: This meta-analysis identifies common methodological pitfalls in LLM security research, providing guidance for more rigorous and reproducible security evaluations.

中文摘要: 这项元分析识别了 LLM 安全研究中常见的方法论陷阱，为更严格和可重现的安全评估提供指导。

10. DUALBREACH: Efficient Dual-Jailbreaking via Target-Driven Initialization and Multi-Target Optimization

Authors: Xinzhe Huang (Zhejiang University), Kedong Xiu, Tianhang Zheng, et al.

English Abstract: DUALBREACH introduces an efficient dual-jailbreaking approach that simultaneously attacks multiple safety mechanisms in LLMs through target-driven initialization and multi-target optimization.

中文摘要: DUALBREACH 引入了一种高效的双重越狱方法，通过目标驱动的初始化和多目标优化同时攻击 LLM 中的多个安全机制。

二、机器学习安全 / ML Security

11. A Deep Dive into Function Inlining and its Security Implications for ML-based Binary Analysis

Authors: Omar Abusabha (Sungkyunkwan University), Jiyong Uhm, Tamer Abuhmed, Hyungjoon Koo

English Abstract: This work investigates how function inlining affects ML-based binary analysis, revealing that compiler optimizations can significantly impact the accuracy and robustness of ML models for malware detection.

中文摘要: 本研究调查了函数内联如何影响基于 ML 的二进制分析，揭示了编译器优化可以显著影响恶意软件检测 ML 模型的准确性和鲁棒性。

12. A Hard-Label Black-Box Evasion Attack against ML-based Malicious Traffic Detection Systems

Authors: Zixuan Liu (Tsinghua University), Yi Zhao (Beijing Institute of Technology), et al.

English Abstract: The paper presents a hard-label black-box evasion attack that can fool ML-based malicious traffic detection systems with minimal queries and no knowledge of the model internals.

中文摘要: 本文提出了一种硬标签黑盒逃避攻击，可以用最少的查询次数和不了解模型内部的情况下欺骗基于 ML 的恶意流量检测系统。

13. Achieving Zen: Combining Mathematical and Programmatic Deep Learning Model Representations for Attribution and Reuse

Authors: David Oygenblik (Georgia Tech), Dinko Dermendzhiev, et al.

English Abstract: Zen combines mathematical and programmatic representations of deep learning models to enable better model attribution and reuse detection, addressing intellectual property concerns in ML.

中文摘要: Zen 结合了深度学习模型的数学和程序表示，以实现更好的模型归属和重用检测，解决了机器学习中的知识产权问题。

14. A Unified Defense Framework Against Membership Inference in Federated Learning

Authors: Liwei Zhang (Beijing University of Posts and Telecommunications), et al.

English Abstract: This work proposes a unified defense framework combining distillation and contribution-aware aggregation to protect against membership inference attacks in federated learning settings.

中文摘要: 本研究提出了一个统一的防御框架，结合蒸馏和贡献感知聚合来防御联邦学习环境中的成员推理攻击。

15. Beyond Raw Bytes: Towards Large Malware Language Models

Authors: Luke Kurlandski (Rochester Institute of Technology), Harel Berger (Ariel University), et al.

English Abstract: This research explores the development of large language models specifically trained on malware data, moving beyond raw byte analysis to understand malicious code semantics.

中文摘要: 本研究探索了专门在恶意软件数据上训练的大型语言模型的开发，超越原始字节分析以理解恶意代码语义。

16. Cascading and Proxy Membership Inference Attacks

Authors: Yuntao Du (Purdue University), Jiacheng Li, et al.

English Abstract: The paper introduces cascading and proxy membership inference attacks that significantly improve attack success rates against machine learning models while requiring less information about the target.

中文摘要: 本文引入了级联和代理成员推理攻击，显著提高了对机器学习模型的攻击成功率，同时需要关于目标的信息更少。

17. CatBack: Universal Backdoor Attacks on Tabular Data via Categorical Encoding

Authors: Behrad Tajalli (Radboud University), Stefanos Koffas (Delft University of Technology), Stjepan Picek

English Abstract: CatBack demonstrates universal backdoor attacks on tabular data through categorical encoding manipulation, affecting a wide range of ML models used in financial and healthcare applications.

中文摘要: CatBack 通过分类编码操纵展示了对表格数据的通用后门攻击，影响金融和医疗应用中使用的广泛 ML 模型。

18. Causal-Guided Detoxify Backdoor Attack of Open-Weight LoRA Models

Authors: Linzhi Chen (ShanghaiTech University), Yang Sun, Hongru Wei, Yuqi Chen

English Abstract: This work presents a causal-guided approach to inject and detoxify backdoors in open-weight LoRA models, revealing vulnerabilities in popular parameter-efficient fine-tuning methods.

中文摘要: 本研究提出了一种因果引导的方法来注入和解毒开放权重 LoRA 模型中的后门，揭示了流行的参数高效微调方法中的漏洞。

19. Constructive Noise Defeats Adversarial Noise: Adversarial Example Detection for Commercial DNN Services

Authors: Meng Shen (Beijing Institute of Technology), Jiangyuan Bi, et al.

English Abstract: The paper proposes using constructive noise to defeat adversarial noise, providing an effective detection mechanism for adversarial examples in commercial DNN services.

中文摘要: 本文提出使用建设性噪声来对抗对抗性噪声，为商业 DNN 服务中的对抗样本提供了有效的检测机制。

20. Dataset Reduction and Watermark Removal via Self-supervised Learning for Model Extraction Attack

Authors: Hao Luan (Fudan University), Xue Tan, Zhiheng Li, et al.

English Abstract: This research demonstrates how self-supervised learning can be leveraged to reduce dataset requirements and remove watermarks in model extraction attacks.

中文摘要: 本研究展示了如何利用自监督学习来减少数据集需求并在模型提取攻击中移除水印。

三、IoT 与嵌入式系统安全 / IoT & Embedded Security

21. ADGFUZZ: Assignment Dependency-Guided Fuzzing for Robotic Vehicles

Authors: Yuncheng Wang (IIE, CAS), Yaowen Zheng, et al.

English Abstract: ADGFUZZ introduces assignment dependency-guided fuzzing specifically designed for robotic vehicles, significantly improving bug discovery in autonomous systems.

中文摘要: ADGFUZZ 引入了专为机器人车辆设计的赋值依赖引导模糊测试，显著提高了自动系统中的错误发现率。

22. An LLM-Driven Fuzzing Framework for Detecting Logic Instruction Bugs in PLCs

Authors: Jiaxing Cheng (IIE, CAS), Ming Zhou (NJUST), Haining Wang (Virginia Tech), et al.

English Abstract: This work leverages LLMs to drive fuzzing for programmable logic controllers (PLCs), detecting critical logic instruction bugs in industrial control systems.

中文摘要: 本研究利用 LLM 驱动可编程逻辑控制器 (PLC) 的模糊测试，检测工业控制系统中的关键逻辑指令错误。

23. FirmAgent: Leveraging Fuzzing to Assist LLM Agents with IoT Firmware Vulnerability Discovery

Authors: Jiangan Ji (Information Engineering University, Tsinghua University), Chao Zhang, et al.

English Abstract: FirmAgent combines fuzzing with LLM agents to enhance IoT firmware vulnerability discovery, bridging the gap between automated testing and intelligent analysis.

中文摘要: FirmAgent 将模糊测试与 LLM 代理相结合，增强物联网固件漏洞发现，弥合自动化测试和智能分析之间的差距。

24. FirmCross: Detecting Taint-style Vulnerabilities in Modern C-Lua Hybrid Web Services

Authors: Runhao Liu (NUDT), Jiarun Dai (Fudan University), et al.

English Abstract: FirmCross addresses the challenge of detecting taint-style vulnerabilities in modern firmware that uses hybrid C-Lua web services, common in router and IoT device firmware.

中文摘要: FirmCross 解决了在使用混合 C-Lua Web 服务的现代固件中检测污点式漏洞的挑战，这在路由器和物联网设备固件中很常见。

25. FlyTrap: Physical Distance-Pulling Attack Towards Camera-based Autonomous Target Tracking Systems

Authors: [Authors truncated in source]

English Abstract: FlyTrap demonstrates physical distance-pulling attacks against camera-based autonomous tracking systems, manipulating perceived distances to compromise safety-critical applications.

中文摘要: FlyTrap 展示了针对基于相机的自主跟踪系统的物理距离拉动攻击，操纵感知距离以危害安全关键应用。

四、密码学与隐私 / Cryptography & Privacy

26. Abuse Resistant Traceability with Minimal Trust for Encrypted Messaging Systems

Authors: Zhongming Wang (Chongqing University), Tao Xiang, et al.

English Abstract: This work proposes an abuse-resistant traceability mechanism for encrypted messaging systems that minimizes trust requirements while enabling accountability.

中文摘要: 本研究为加密消息系统提出了一种抗滥用的可追溯机制，在实现问责的同时最小化信任需求。

27. BKPIR: Keyword PIR for Private Boolean Retrieval

Authors: Jie Song (IIE, CAS), Zhen Xu, et al.

English Abstract: BKPIR introduces an efficient keyword-based private information retrieval scheme specifically designed for private Boolean retrieval operations.

中文摘要: BKPIR 引入了一种专为私有布尔检索操作设计的高效基于关键字的私有信息检索方案。

28. Cryptobazaar: Private Sealed-bid Auctions at Scale

Authors: Andrija Novakovic (Bain Capital Crypto), Alireza Kavousi (UCL), et al.

English Abstract: Cryptobazaar enables private sealed-bid auctions at scale using cryptographic techniques, ensuring bid privacy while maintaining auction integrity.

中文摘要: Cryptobazaar 使用密码学技术实现大规模私有密封投标拍卖，确保投标隐私同时保持拍卖完整性。

29. CryptPEFT: Efficient and Private Neural Network Inference via Parameter-Efficient Fine-Tuning

Authors: Saisai Xia (IIE, CAS), Wenhao Wang, et al.

English Abstract: CryptPEFT achieves efficient and private neural network inference by combining cryptographic techniques with parameter-efficient fine-tuning methods.

中文摘要: CryptPEFT 通过将密码学技术与参数高效微调方法相结合，实现高效且私密的神经网络推理。

30. cwPSU: Efficient Unbalanced Private Set Union via Constant-weight Codes

Authors: Qingwen Li (Xidian University), Song Bian (Beihang University), Hui Li

English Abstract: cwPSU proposes an efficient solution for unbalanced private set union using constant-weight codes, significantly improving performance over existing approaches.

中文摘要: cwPSU 提出了一种使用常权码进行不平衡私有集合并集的高效解决方案，显著提高了现有方法的性能。

五、Web 与网络安全 / Web & Network Security

31. Achieving Interpretable DL-based Web Attack Detection through Malicious Payload Localization

Authors: Peiyang Li (Tsinghua & Ant Group), Fukun Mei, et al.

English Abstract: This work achieves interpretable deep learning-based web attack detection by localizing malicious payloads, enabling security analysts to understand and verify detection decisions.

中文摘要: 本研究通过定位恶意载荷实现可解释的基于深度学习的 Web 攻击检测，使安全分析师能够理解和验证检测决策。

32. AirSnitch: Demystifying and Breaking Client Isolation in Wi-Fi Networks

Authors: Xin’an Zhou (UC Riverside), Juefei Pu, et al.

English Abstract: AirSnitch reveals vulnerabilities in Wi-Fi client isolation mechanisms, demonstrating practical attacks that break the security boundary between isolated clients.

中文摘要: AirSnitch 揭示了 Wi-Fi 客户端隔离机制中的漏洞，展示了打破隔离客户端之间安全边界的实际攻击。

33. Aliens Among Us: Observing Private or Reserved IPs on the Public Internet

Authors: Radu Anghel (TU Delft), Carlos Gañán (ICANN), et al.

English Abstract: This measurement study observes the prevalence of private and reserved IP addresses appearing on the public Internet, analyzing their security implications.

中文摘要: 这项测量研究观察了私有和保留 IP 地址在公共互联网上出现的普遍性，分析其安全影响。

34. CHAMELEOSCAN: Demystifying and Detecting iOS Chameleon Apps via LLM-Powered UI Exploration

Authors: Hongyu Lin (Zhejiang University), Yicheng Hu, et al.

English Abstract: CHAMELEOSCAN uses LLM-powered UI exploration to detect iOS chameleon apps that change behavior after passing App Store review.

中文摘要: CHAMELEOSCAN 使用 LLM 驱动的 UI 探索来检测在通过 App Store 审核后改变行为的 iOS 变色龙应用。

35. DOM-XSS Detection via Webpage Interaction Fuzzing and URL Component Synthesis

Authors: Nuno Sabino (CMU, IST Lisbon), Darion Cassel, et al.

English Abstract: This work proposes a novel approach to detect DOM-based XSS vulnerabilities through automated webpage interaction fuzzing and intelligent URL component synthesis.

中文摘要: 本研究提出了一种通过自动化网页交互模糊测试和智能 URL 组件合成来检测基于 DOM 的 XSS 漏洞的新方法。

总结 / Summary

NDSS 2026 展示了网络安全研究的最新进展，涵盖了 LLM 安全、机器学习安全、IoT 安全、密码学和 Web 安全等多个领域。特别值得关注的趋势包括：

  1. LLM 安全成为主流话题：从越狱攻击到防御机制，从提示注入到水印破坏，LLM 安全研究已经成为顶级安全会议的重要组成部分。

  2. 机器学习安全持续深入：研究不仅关注对抗样本和模型窃取，还深入到联邦学习、成员推理和后门攻击等更复杂的场景。

  3. IoT 与嵌入式系统安全受到重视：随着自动驾驶、工业控制系统和智能设备的普及，相关安全研究变得越来越重要。

  4. 密码学与隐私保护技术创新：私有信息检索、安全多方计算和隐私保护机器学习等技术持续演进。

  5. 自动化与智能化分析工具：LLM 被广泛应用于安全分析、漏洞发现和代码审计等领域。

NDSS 2026 represents the cutting edge of network and distributed system security research. Key trends include:

  1. LLM Security as Mainstream: From jailbreak attacks to defense mechanisms, LLM security has become a major focus of top-tier security conferences.

  2. Deep ML Security Research: Beyond adversarial examples and model extraction, research now covers federated learning, membership inference, and backdoor attacks.

  3. IoT & Embedded Security: Critical as autonomous vehicles, industrial control systems, and smart devices proliferate.

  4. Cryptographic & Privacy Innovations: PIR, secure multi-party computation, and privacy-preserving ML continue to advance.

  5. Automated & Intelligent Analysis: LLMs are widely adopted for security analysis, vulnerability discovery, and code auditing.
