NDSS 2026 论文精选 - 系统安全视角

从系统安全视角看 NDSS 2026: 涵盖操作系统、内核、虚拟化、内存安全、二进制分析

Perspective: System Security | Focus: OS, Kernel, Binary, Memory, Virtualization

一、操作系统与内核安全

1. DirtyFree: Simplified Data-Oriented Programming in Linux Kernel

Authors: Yoochan Lee (Max Planck), Hyuk Kwon (Theori), Thorsten Holz

English: Simplifies data-oriented programming attacks in Linux kernel, demonstrating practical exploitation without control-flow hijacking.

中文: 简化 Linux 内核中的数据导向编程攻击，演示无需控制流劫持的实际利用。

2. Cross-Cache Attacks via PCP Massaging

Authors: Claudio Migliorelli et al. (IBM Research Europe)

English: Novel cross-cache attacks on Linux kernel via per-CPU pages (PCP) massaging, bypassing traditional cache isolation.

中文: 通过每 CPU 页面 (PCP) 按摩对 Linux 内核的新型跨缓存攻击，绕过传统缓存隔离。

3. Demystifying ESXi VMKernel Access Control Mechanism

Authors: Yue Liu et al. (Southeast University, NUDT, Zhejiang University)

English: In-depth analysis of ESXi VMKernel’s access control, revealing design weaknesses and potential attack surfaces.

中文: 深入分析 ESXi VMKernel 的访问控制，揭示设计弱点和潜在攻击面。

4. Eviction Notice: Reviving Page Cache Attacks

Authors: Sudheendra Neela et al. (Graz University of Technology)

English: Revives and advances page cache attacks with new techniques applicable to modern operating systems.

中文: 用适用于现代操作系统的新技术复兴和推进页缓存攻击。

二、虚拟化与隔离

5. Breaking Isolation: Cross-Domain Hypervisor Attacks

Authors: Gaoning Pan et al. (Hangzhou Dianzi University, Zhejiang University)

English: New perspective on hypervisor exploitation via cross-domain attacks, breaking VM isolation boundaries.

中文: 通过跨域攻击对虚拟机管理程序利用的新视角，打破虚拟机隔离边界。

6. Bit of a Close Talker: Serverless Cloud Co-Location Attacks

Authors: Wei Shao et al. (UC Davis, George Mason University)

English: Practical guide to serverless cloud co-location attacks, demonstrating side-channel vulnerabilities in AWS Lambda and similar platforms.

中文: 无服务器云共处攻击的实用指南，演示 AWS Lambda 和类似平台中的侧信道漏洞。

7. EXIA: Trusted Transitions for Enclaves via External-Input Attestation

Authors: Zhen Huang et al. (Shanghai Jiao Tong University, Auburn University)

English: Ensures trusted transitions for secure enclaves through external-input attestation mechanisms.

中文: 通过外部输入证明机制确保安全区的可信转换。

8. Exploiting TLBs in Virtualized GPUs for Cross-VM Attacks

Authors: Hongyue Jin et al. (Clemson University, University of Rochester)

English: Exploits TLB vulnerabilities in virtualized GPUs to perform cross-VM side-channel attacks.

中文: 利用虚拟化 GPU 中的 TLB 漏洞执行跨虚拟机侧信道攻击。

三、二进制分析与逆向工程

9. A Deep Dive into Function Inlining for ML-based Binary Analysis

Authors: Omar Abusabha et al. (Sungkyunkwan University)

English: Investigates how function inlining affects ML-based binary analysis, revealing compiler optimization impacts on malware detection.

中文: 调查函数内联如何影响基于 ML 的二进制分析，揭示编译器优化对恶意软件检测的影响。

10. BINALIGNER: Cross-Compilation Environment Diffing

Authors: Yiran Zhu et al. (Zhejiang University, UCL)

English: Aligns binary code across different compilation environments to enable accurate binary diffing and analysis.

中文: 跨不同编译环境对齐二进制代码以实现准确的二进制差异和分析。

11. Enhancing Semantic-Aware Binary Diffing

Authors: Chengfan Ye et al. (HKUST)

English: Enhances binary diffing with high-confidence dynamic instruction alignment for better semantic understanding.

中文: 通过高置信度动态指令对齐增强二进制差异以更好地理解语义。

12. FidelityGPT: Correcting Decompilation Distortions

Authors: Zhiping Zhou et al. (Tianjin University, Southern Cross University)

English: Uses retrieval-augmented generation to correct decompilation distortions, improving code reconstruction fidelity.

中文: 使用检索增强生成纠正反编译失真，提高代码重建保真度。

四、内存安全

13. Fast Pointer Nullification for Use-After-Free Prevention

Authors: Yubo Du et al. (University of Pittsburgh)

English: Fast pointer nullification mechanism to prevent use-after-free vulnerabilities with minimal performance overhead.

中文: 快速指针归零机制以最小性能开销防止释放后使用漏洞。

14. FLIPPYRAM: Large-Scale Rowhammer Prevalence Study

Authors: Martin Heckel et al. (Hof University, Graz University)

English: Large-scale study of Rowhammer vulnerability prevalence across different DRAM modules and systems.

中文: 跨不同 DRAM 模块和系统的 Rowhammer 漏洞流行度大规模研究。

五、可信执行环境 (TEE)

15. Automated Code Annotation with LLMs for Establishing TEE Boundaries

Authors: Varun Gadey et al. (University of Würzburg, Huawei)

English: Uses LLMs to automate code annotation for establishing Trusted Execution Environment boundaries.

中文: 使用 LLM 自动化代码注释以建立可信执行环境边界。

研究趋势

🔍 核心主题

  1. 数据导向攻击: 从控制流到数据流的攻击转移
  2. 跨域攻击: 打破虚拟化、容器、云环境的隔离边界
  3. 侧信道进化: TLB、页缓存、GPU 等新型侧信道
  4. 二进制分析现代化: AI/LLM 辅助的反编译和分析
  5. 内存安全持续关注: Rowhammer、UAF 等经典问题的新解决方案

💡 技术洞察

  • 从控制流到数据流: DOP (Data-Oriented Programming) 成为主流攻击方式
  • 隔离失效: 虚拟机、容器、serverless 的隔离机制被持续挑战
  • AI 赋能逆向: LLM 应用于反编译、二进制分析、代码重建
  • 硬件漏洞持久性: Rowhammer 等硬件级漏洞难以根除

🛠️ 实际影响

  • 云安全: 跨虚拟机、serverless 攻击威胁云服务提供商
  • 企业系统: Linux 内核、VMware ESXi 等企业基础设施的安全
  • 安全审计: 二进制分析工具的准确性和效率提升
  • 内存安全: UAF、Rowhammer 防护对关键系统至关重要
